발표자료(PPT) 만들기

⏰ 오후 2:00 ~ 5:00 📍 작은행동한사랑 3층 교육장

가지고 있는 안내문·보고서·모임 자료를 발표자료로 바꿉니다. 발표자료는 ‘글을 옮기는 것’이 아니라 ‘핵심을 보여주는 것’이에요.

이 시간에 만드는 결과물 미리보기
발표 제목2026 · 발표자료

학습 목표

  • 기존 문서(워드·PDF 등)를 발표자료로 바꾼다.
  • 줄글·목록을 보기 좋은 슬라이드로 정리한다.
  • 발표자료를 내보내 실제로 사용한다.

핵심 내용

  • 기존 문서를 올려 발표자료로 변환하는 흐름
  • 한 슬라이드 = 한 메시지 원칙, 글자 줄이고 핵심만
  • 목록을 다이어그램·도표 같은 시각 요소로 바꾸기
  • 내보내기 형식 안내(상황에 따라 변환이 필요할 수 있음을 함께 설명)

실습 활동

예시 프롬프트

“이 문서를 10장짜리 발표자료로 만들어줘. 한 장에 핵심 3줄 이내로, 목록은 가능하면 도표나 다이어그램으로 바꿔줘. 차분하고 깔끔한 느낌으로.”

→ “3번 슬라이드를 표로 바꿔줘”처럼 슬라이드별 수정.

결과물

기존 문서 기반 발표자료 1편(8~12장)

강사 유의사항 · 팁

  • 수강생이 가진 안내문·보고서·모임 자료를 재료로 쓰면 실용성이 높음
  • 발표자료는 ‘글을 옮기는 것’이 아니라 ‘핵심을 보여주는 것’임을 반복 강조
보충 자료

기존 문서를 PPT로 만들기

이미 가진 문서를 발표자료(PPT)로 바꾸는 길은 크게 세 가지예요. “어떤 도구로, 어떤 문서를, 어떻게”를 나눠서 보면 깔끔합니다. 일반인 실습은 경로 1만 메인으로 잡고, 2·3은 “더 전문적으로 하면 이런 것도 된다”로 소개하면 부담이 적어요.

① 어떤 문서를 넣을 수 있나요?

docx워드 — 텍스트 구조 그대로 변환
pdfPDF — 내용을 읽어 슬라이드로
ppt기존 덱을 직접 읽어 재구성
img이미지 속 내용을 슬라이드 소재로
  • docx · pdf — 텍스트 구조를 그대로 읽어 슬라이드로 변환 (가장 잘 됩니다)
  • ppt — 경로 2(애드인)에서 기존 덱을 직접 읽어 편집·재구성
  • 이미지(img) — 도표·사진 캡처 등 이미지 속 내용을 읽어 슬라이드 소재로 활용

② 텍스트 → 디자인 텍스트 (핵심!)

여기가 가장 인상적인 부분이에요. 단순히 글자를 옮기는 게 아니라 줄글·불릿을 시각 요소로 바꿔준다는 점입니다. 슬라이드·차트·도형이 네이티브 파워포인트 객체로 생성되어, 다른 슬라이드처럼 자유롭게 수정할 수 있어요.

변환 — 줄글이 디자인된 슬라이드로
기존 문서 (줄글)
→
디자인된 슬라이드
1텍스트 → 디자인 텍스트 — 밋밋한 줄글을 제목·소제목·강조 위계가 잡힌 슬라이드 문구로 재구성
2텍스트+이미지 → 디자인된 텍스트+이미지 — 불릿은 다이어그램·차트로, 단순 나열은 카드·타임라인 레이아웃으로 배치

이렇게 요청해 보세요

“이 불릿들을 프로세스 플로우 다이어그램으로 바꿔줘”
“Q1과 Q4를 비교하는 막대그래프를 만들어줘”

③ 만드는 세 가지 경로

경로 1클로드 채팅에서 .pptx 만들기 — 가장 쉽고 범용

문서를 채팅창에 올리고 “PPT로 만들어줘”라고 하면 됩니다. 클로드가 원본의 텍스트 구조를 유지하며 슬라이드를 작성하고, 코드를 실행해 파일을 만든 뒤 다운로드 링크를 대화창에 올려줍니다.

사전 준비

설정(Settings) → 기능(Capabilities) → ‘코드 실행 및 파일 생성’을 켜두세요. 모든 유료 플랜에서 .pptx 생성이 가능합니다.

경로 2클로드 for PowerPoint 애드인 — 내 템플릿 그대로 살릴 때

파워포인트 안에서 직접 작동합니다. 기존 덱의 레이아웃·폰트·색상·슬라이드 마스터를 읽어 내 서식을 존중하며 편집해줘요. 회사·학교 템플릿을 유지한 채 변환하거나, 특정 슬라이드만 콕 집어 수정할 수 있습니다. 웹·윈도우·맥 파워포인트에서 쓸 수 있고 유료 플랜(Pro·Max·Team·Enterprise)이 필요합니다.

경로 3클로드 디자인 — 시각적으로 화려한 덱

디자인 중심 덱에 좋지만 주의점이 있어요. 결과물이 HTML 파일로 나와서, .pptx나 구글 슬라이드처럼 평소 프로그램에서 바로 열려면 변환이 필요합니다. 또 회사 브랜드 PPT 템플릿을 업로드해 쓰는 기능은 없습니다.

④ 추천 실습 흐름

1변환할 문서 1개 준비 (docx 또는 pdf)
2클로드 채팅에 올리고 이렇게 요청: “이 문서를 12장짜리 발표자료로 만들어줘. 핵심만 슬라이드당 3줄 이내로, 불릿은 가능하면 다이어그램으로. .pptx로 내보내줘”
3나온 파일을 파워포인트에서 열어 확인
4더 다듬고 싶으면 → 경로 2(애드인)로 슬라이드별 미세 수정
